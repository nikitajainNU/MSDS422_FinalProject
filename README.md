# NFL Team Win Prediction

## Overview
This project aims to predict whether an NFL team will win a game based on historical game statistics and team performance data. It is a binary classification problem.

## Objectives
- Predict NFL game outcomes
- Identify important features
- Compare machine learning models

## Dataset
The dataset includes historical NFL game-level or team-level statistics, such as offensive and defensive performance, turnovers, home/away status, and other relevant variables. The target variable indicates whether the team won the game.

## Project Structure
- `data/`: raw and processed datasets
- `notebooks/`: notebooks for EDA, feature engineering, and modeling
- `src/`: scripts for preprocessing, training, and evaluation
- `results/`: plots, metrics, and prediction outputs

## Methodology
Our workflow includes data cleaning, exploratory data analysis, feature engineering, model development, and model evaluation.

## Modeling and Evaluation
We plan to begin with a baseline model and then compare more advanced classification models such as logistic regression, random forest, and gradient boosting. Performance will be evaluated using accuracy, precision, recall, F1-score, and ROC-AUC.

## Current Progress
- Data collection and cleaning completed
- Initial EDA and Feature engineering completed
- Modeling and evaluation in progress

## How to Run
1. Install required packages and set up environment
2. Open the notebooks in the `notebooks/` folder
3. Run the preprocessing, modeling, and evaluation steps
